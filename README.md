# Mathemagician v3

Jogo educativo infantil de matemática com estética mágica.

## Descrição
Aplicação/jogo infantil de matemática com 4 modos de jogo, progressão por reinos e itens mágicos para desbloquear.

## Modos de Jogo
- **Aprender a Contar** - Contagem de 1 até X com texto por extenso
- **Touchculator** - Operações visuais com círculos
- **Modo História** - 4 reinos, 16 níveis, 80 perguntas
- **Modo Livre** - 10 perguntas à escolha do jogador

## Requisitos
- Node.js 18+
- npm 9+

## Instalação
```bash
npm install
```

## Desenvolvimento
```bash
npm start
```

## Construção
```bash
npm run build
```
